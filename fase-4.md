# Fase 4 — Polish & Deploy

## Doel

De app productie-waardig maken en live zetten op een publieke URL.

## Wat

- Responsive check: alles werkt goed op mobiel, tablet en desktop
- Loading states: de gebruiker ziet dat data wordt opgehaald
- Foutafhandeling: als de scraper faalt, krijgt de gebruiker geen errors te zien
- Visuele polish: spacing, hover states, transities waar nodig
- Vercel deployment configuratie
- README met wat de app doet en hoe te deployen

## Klaar wanneer

De app draait live op een Vercel URL. Het ziet er professioneel uit, werkt op alle schermformaten, en handelt fouten graceful af.
